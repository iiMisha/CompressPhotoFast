# Формат: один токен на строку. Строки, начинающиеся с '#', и пустые — игнорируются.
# Скрипты find_unused.py / find_duplicates.py и LLM-анализ читают этот файл.

# как unused, но фактически используются (Android lifecycle, DI, синтаксис).
