CompressPhotoFast — кроссплатформенное приложение для сжатия фотографий. Android-версия (API 29+, Kotlin, MVVM, Hilt, WorkManager) и CLI-инструмент (Python 3.10+). Идентичная логика сжатия с EXIF-маркерами. CLI поддерживает многопроцессорную обработку (ProcessPoolExecutor) и HEIC/HEIF (pillow-heif). Android: 232 теста (50 unit + 182 instrumentation), JaCoCo coverage ~8-10%. Недавние оптимизации: устранены Context Leaks, заменен Handler на корутины, внедрен Mutex для синхронизации.
