CompressPhotoFast — это кроссплатформенное приложение для быстрого и эффективного сжатия фотографий. Включает Android-приложение (API 29+) и CLI-инструмент для Linux/Windows. Android-версия использует Kotlin с современными архитектурными компонентами (Hilt, WorkManager, MVVM), а CLI-версия написана на Python 3.10+. Обе версии используют идентичную логику сжатия с EXIF-маркерами для отслеживания сжатых файлов. CLI-версия поддерживает многопроцессорную обработку с ProcessPoolExecutor и multiprocessing.Manager().Lock() для process-safe операций, а также поддержку HEIC/HEIF форматов через pillow-heif.
