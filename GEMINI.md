# Проект: CompressPhotoFast

## Обзор проекта

Это Android-приложение под названием **CompressPhotoFast**, предназначенное для быстрого и эффективного сжатия фотографий с сохранением оптимального качества изображения. Приложение написано на Kotlin и использует современные практики разработки под Android.

Проект имеет стандартную структуру Android-приложения с одним модулем `app`. Для сборки используется Gradle с Kotlin DSL. Ключевые особенности включают пакетную обработку изображений, настраиваемые параметры сжатия, фоновую обработку и поддержку различных форматов изображений (JPEG, PNG, WEBP, HEIC/HEIF).

## Сборка и запуск

Для сборки и запуска этого проекта вам понадобится Android Studio.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/yourusername/CompressPhotoFast.git
    ```

2.  **Откройте в Android Studio:**
    Откройте клонированный каталог в Android Studio. Проект должен автоматически синхронизироваться с Gradle.

3.  **Сборка приложения:**
    Вы можете собрать приложение, используя следующие команды Gradle из корневого каталога проекта:

    *   **Debug-сборка:**
        ```bash
        ./gradlew assembleDebug
        ```

    *   **Release-сборка:**
        ```bash
        ./gradlew assembleRelease
        ```
        Release APK будет назван `CompressPhotoFast_v<versionName>.apk` и будет находиться в каталоге `app/build/outputs/apk/release/`.

4.  **Запуск приложения:**
    Запустите приложение на эмуляторе или физическом устройстве прямо из Android Studio.

5.  **Запуск тестов:**
    Для запуска юнит-тестов выполните следующую команду:
    ```bash
    ./gradlew test
    ```
    Для запуска инструментальных тестов выполните:
    ```bash
    ./gradlew connectedAndroidTest
    ```

## Соглашения о разработке

*   **Язык:** Проект полностью написан на Kotlin.
*   **Архитектура:** Используются компоненты архитектуры Android, включая ViewModel, LiveData и WorkManager.
*   **Внедрение зависимостей:** Для внедрения зависимостей используется Dagger Hilt.
*   **Асинхронное программирование:** Для обработки асинхронных операций используются корутины Kotlin и Flow.
*   **Обработка изображений:**
    *   Coil используется для загрузки и отображения изображений.
    *   Compressor используется для сжатия изображений.
    *   ExifInterface используется для чтения и записи метаданных изображений.
*   **Настройки:** DataStore используется для хранения настроек приложения.
*   **Логирование:** Для логирования используется Timber.
*   **Пользовательский интерфейс:** Пользовательский интерфейс создан с использованием компонентов Material Design 3.
*   **Тестирование:**
    *   Юнит-тесты написаны на JUnit 4.
    *   Инструментальные тесты написаны на Espresso.
*   **Стиль кода:** Проект следует стандартным соглашениям о кодировании на Kotlin.
*   **Версионирование:** Имя версии в `app/build.gradle.kts` соответствует формату `X.Y.Z(dd.MM.yyyy)`.
