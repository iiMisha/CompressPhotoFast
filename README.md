# CompressPhotoFast

CompressPhotoFast - это Android-приложение для автоматического и ручного сжатия фотографий, которое помогает экономить место на устройстве, сохраняя при этом приемлемое качество изображений.

## Основные возможности

### 1. Автоматическое сжатие
- Автоматическое обнаружение новых фотографий на устройстве
- Фоновое сжатие изображений каждые 15 минут
- Работа даже при выгруженном приложении благодаря использованию JobScheduler

### 2. Ручное сжатие
- Возможность выбора конкретных изображений для сжатия
- Предварительный просмотр фотографий
- Выбор степени сжатия

### 3. Поддерживаемые форматы
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

### 4. Дополнительные функции
- Сохранение EXIF-данных после сжатия
- Создание отдельной папки для сжатых изображений
- Возможность настройки параметров сжатия
- Отображение статистики по сжатию (размер до/после)

## Как использовать

### Автоматическое сжатие

1. Откройте приложение
2. Включите опцию автоматического сжатия в настройках
3. Приложение будет автоматически обнаруживать и сжимать новые фотографии

### Ручное сжатие

1. Откройте приложение
2. Нажмите кнопку "Выбрать фото"
3. Выберите одно или несколько изображений
4. Настройте параметры сжатия (если необходимо)
5. Нажмите "Сжать"

## Настройки

- **Автоматическое сжатие**: включение/выключение автоматического режима
- **Качество сжатия**: настройка степени сжатия (влияет на качество итогового изображения)
- **Сохранение EXIF**: включение/выключение сохранения метаданных
- **Папка назначения**: выбор места сохранения сжатых изображений

## Системные требования

- Android 5.0 (API 21) и выше
- Разрешение на доступ к файлам
- Достаточно свободного места для временных файлов

## Безопасность

- Приложение не удаляет оригинальные фотографии
- Сжатые копии сохраняются в отдельной папке
- Все операции можно отменить

## Ограничения

- Время обработки зависит от размера изображения и мощности устройства
- В режиме энергосбережения автоматическое сжатие может выполняться с задержкой
- Некоторые форматы изображений могут быть не поддержаны
