# CompressPhotoFast

Приложение для автоматического сжатия фотографий на Android.

## Описание

CompressPhotoFast - это простое и эффективное приложение для сжатия фотографий на устройствах Android. Приложение позволяет сжимать как выбранные вручную изображения, так и автоматически обрабатывать новые фотографии, появляющиеся в галерее.

## Основные функции

- Сжатие выбранных изображений
- Автоматическое сжатие новых фотографий
- Сохранение метаданных (EXIF) при сжатии
- Сохранение оригинальных имен файлов с добавлением суффикса "_compressed"
- Возможность получать изображения для сжатия из других приложений
- Автоматический запуск при загрузке системы

## Технический стек

- Kotlin
- MVVM архитектура
- Hilt для внедрения зависимостей
- WorkManager для фоновых задач
- Compressor для сжатия изображений
- ExifInterface для сохранения метаданных
- DataStore для хранения настроек
- Timber для логирования

## Требования

- Android 10 (API 29) и выше
- Разрешения на доступ к хранилищу и отправку уведомлений

## Установка

1. Клонировать репозиторий
2. Открыть проект в Android Studio
3. Собрать и установить приложение на устройство

## Использование

1. Запустите приложение
2. Выберите изображение для сжатия с помощью кнопки "Выбрать изображение"
3. Нажмите кнопку "Сжать фото" для обработки выбранного изображения
4. Включите переключатель "Автоматическое сжатие" для активации фонового мониторинга новых фотографий
