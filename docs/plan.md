# План задач

## Новая функциональность
- Добавить переключатель на главном экране приложения: - готово
  - При включённом переключателе сжатые файлы записываются в исходную папку, а оригинальные файлы удаляются.
  - При выключенном переключателе сжатые файлы сохраняются в отдельной папке, а оригиналы остаются.

- В перспективе разработать механизм принятия решения о сжатии файла с использованием библиотеки AIMTools

## Исправление ошибок
- Устранение дублирования файлов при получении через "поделиться" - исправлено

- В момент включения "Автоматическое сжатие", если до этого было активировано "Заменять оригинальные файлы", создается лишний дубликат сжатого файла
