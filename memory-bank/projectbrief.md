# PROJECT BRIEF - CompressPhotoFast

**Проект:** CompressPhotoFast  
**Тип:** Android приложение  
**Платформа:** Android (минимум API 24)  
**Язык:** Kotlin  
**Дата создания:** $(Get-Date -Format "yyyy-MM-dd")  

## ОБЗОР ПРОЕКТА

CompressPhotoFast - это Android приложение для автоматического сжатия фотографий с сохранением метаданных EXIF. Приложение работает в фоновом режиме и автоматически обрабатывает новые фотографии.

## ОСНОВНЫЕ ЦЕЛИ

1. **Автоматическое сжатие** - Сжимать новые фотографии без участия пользователя
2. **Сохранение метаданных** - Полностью сохранять EXIF данные
3. **Фоновая работа** - Работать в background без влияния на UX
4. **Простота использования** - Минимальный интерфейс, максимальная автоматизация

## ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### Функциональные требования
- Мониторинг новых фотографий в галерее
- Автоматическое сжатие с настраиваемым качеством
- Сохранение всех EXIF метаданных
- Замена оригинального файла сжатой версией
- Уведомления о процессе сжатия

### Нефункциональные требования
- Минимальное потребление батареи
- Быстрая обработка изображений
- Стабильная работа в фоне
- Совместимость с Android 7.0+

## АРХИТЕКТУРА

### Основные компоненты
1. **MainActivity** - Главный экран с настройками
2. **PhotoMonitorService** - Фоновый сервис мониторинга
3. **ImageCompressor** - Утилита сжатия с сохранением EXIF
4. **MediaStoreObserver** - Наблюдатель за новыми фото
5. **CompressionWorker** - WorkManager для фоновой обработки

### Технологический стек
- **Kotlin** - Основной язык разработки
- **AndroidX** - Современные компоненты Android
- **WorkManager** - Фоновые задачи
- **ExifInterface** - Работа с метаданными
- **Foreground Service** - Надежная фоновая работа
