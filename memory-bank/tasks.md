### ОПИСАНИЕ
Инициализация Android приложения для автоматического сжатия фотографий с сохранением метаданных.

### ТРЕБОВАНИЯ
- Автоматическое сжатие новых фотографий
- Сохранение метаданных EXIF
- Фоновая работа
- Простой интерфейс

**Уровень:** 3-4 (ВЫСОКАЯ СЛОЖНОСТЬ)
